# Ublio - Luxury Audio E-Commerce Platform

A premium e-commerce website for luxury audio equipment built with modern web technologies.

## Project Structure

```
UBLIO/
├── frontend/          # Client-side application
│   ├── src/           # Source files
│   ├── public/        # Static assets
│   ├── index.html     # Main entry point
│   └── package.json   # Frontend dependencies
├── backend/           # Server-side API (coming soon)
└── README.md          # This file
```

## Frontend

The frontend is a modern Vite-powered application featuring:
- 43 premium audio products across 9 categories
- Dark/light theme toggle
- Real-time shopping cart
- Responsive design
- SEO optimized

### Quick Start

```bash
cd frontend
npm install
npm run dev
```

Visit: `http://localhost:5173`

### Build for Production

```bash
cd frontend
npm run build
```
